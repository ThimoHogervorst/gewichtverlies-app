# Gewicht Tracker App

Een simpele, snelle web app om je gewichtsverloop te tracken. Perfect voor je iPhone!

## Features

✅ **Gewicht invoeren** - Voeg je dagelijkse metingen toe  
✅ **Doelgewicht** - Stel een doelgewicht in en track je voortgang  
✅ **Grafieken** - Zie je gewichtscurve met een mooie grafiek  
✅ **Statistieken** - Totaal afgevallen, afname per week, etc.  
✅ **Lokale opslag** - Al je data wordt opgeslagen op je telefoon (offline)  
✅ **Geen login** - Geen inloggen nodig, direct gebruiken  

## Hoe te gebruiken

1. Open de app op je iPhone via Safari
2. Voer je huidige datum en gewicht in
3. Stel een doelgewicht in (optioneel)
4. Zie je voortgang in grafieken en statistieken

## Hoe op te zetten op GitHub Pages

1. **Maak een GitHub repository** aan
   - Noem het `gewichtverlies-app` (of wat je wilt)

2. **Upload deze files** naar de repository:
   - `index.html`
   - `style.css`
   - `script.js`

3. **Zet GitHub Pages aan**
   - Ga naar Settings → Pages
   - Kies "Deploy from a branch"
   - Selecteer `main` branch en `/root` folder

4. **Je app is live!**
   - Zou beschikbaar zijn op: `https://[jouw-username].github.io/gewichtverlies-app/`

## Installeren op je iPhone

1. Open de link in Safari
2. Tik op het share-icoontje
3. Selecteer "Voeg toe aan startscherm"
4. Nu staat de app als icon op je home screen!

## Privacy

- Alle data wordt opgeslagen in je browser (localStorage)
- Niets gaat naar een server
- Je gegevens zijn alleen op je toestel

## Customization

Wil je iets aanpassen? Alle code is eenvoudig en goed gecommentarieerd. Je kan bijvoorbeeld:
- De kleuren aanpassen in `style.css`
- Het doelgewicht default instellen in `script.js`
- Extra functies toevoegen

Veel sterkte met je gezondheidsgoals! 💪
